website portfolio one page gratis

Donde registrarte

Registrate en https://gitlab.com/ (configura tu cuenta) y posteriormente en https://vercel.com/new/, conecta tu gitlab con vercel, recuerda usar una contraseña sencilla y que no uses en otros sitios

Commandos para tirar en consola de linux

  • escribe git en tu kali linux o instalalo si es otra distro

  • cd /home/"distro"/Desktop
    ls -l
    touch index.html
    cat index.html
    code index.html
    - escribimos el codigo y lo guardamos al terminar
    cat index.html
    git init
    git add index.html
    git config --global user.email "usuario@tucorreo.com"
    git config --global user.name "usuario"
    git commit -m "mi paginaweb primer push"
    git push

  • tu html se habra subido a tu gitlab.
  • al escribir code se abrira visual code estudio, escribir "y" para instalaro en caso de ser necesario.

Ve a la pagina de gitlab

  • asegurate de que el contenido este igual que en este github, necesitaras: pack.json, vercel.json, index.html, .vercelignore
  • puedes crear notas de texto en gitlab que automaticamente adoptaran el lenguaje que se especifique (python c...)
  • asi es como quedaria

Deploy! en vercel

  • ve a vercelhttps://vercel.com/new/
  • y haz click en tu gitlab conectado a la cuenta, busca la carpeta con pagina web o portfolio
  • clickea en debloy y se subira automaticamente
  • para acceder a ella en el dashboard vendra el link con el dominio " vercel app", enorabuena ya tienes tu pagina web gratis
